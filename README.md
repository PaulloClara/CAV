# CAV
A Clara Assistente Virtual é um projeto relativamente simples que fiz para brincar com
a minha namorada

## O projeto usa:
```
Python3
gtts
SQLite3
cvlc
```

### Para instalar todas as dependências (com um env ativado de preferência), execute esse comando:
```
pip3 install -r dep.txt
```

### Para executar a CAV:
```
python3 -B run.py
```
